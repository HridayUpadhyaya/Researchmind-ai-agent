# ResearchMind AI

An autonomous AI research agent that:

- Plans research tasks
- Searches the web
- Extracts information
- Compares evidence
- Detects contradictions
- Generates reports
- Provides confidence scores

## Tech Stack

- Python
- FastAPI
- React
- DuckDuckGo Search
- BeautifulSoup

## Features

- Multi-step AI workflow
- Live web search
- Source extraction
- Evidence comparison
- Citation-backed reporting
- Confidence scoring

## Architecture

User Query
→ Research Planner
→ Search Engine
→ Source Analyzer
→ Evidence Comparator
→ Report Generator
